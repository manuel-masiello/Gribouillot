# Legacy binaries (version 2.0, from the upstream project)

The binaries in this folder were **not** built from this repository. They are the
**version 2.0 binaries produced by the original upstream project**
([Fr3nchK1ss/Gribouillot](https://github.com/Fr3nchK1ss/Gribouillot), by Ludovic A.),
built with Qt5 around 2018-2020:

- `Gribouillot_2.0_linux64.AppImage` — Linux 64 bits
- `Gribouillot_2.0_win32.zip` — Windows 32 bits
- `linux_integration/` — desktop integration files used for the 2.0 AppImage

They are kept for reference and for users who need the historical version. They do
**not** contain the features and fixes added by this fork (Qt6 port, ellipse tool, ...).

**Up-to-date binaries** for Linux, Windows and macOS are built automatically from the
sources by GitHub Actions and published on the
[Releases page](https://github.com/manuel-masiello/Gribouillot/releases).
